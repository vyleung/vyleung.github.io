About the project: ↗ ClassInSight is a 5-year design-based research project funded by the James S. McDonnell Foundation that studies how secondary science teachers facilitate rich discussions. This collaborative project between Carnegie Melleon University, Penn State University, and UC San Diego aims to create a web application that provides teachers with visualizations of their classroom discussions to help them reflect on their practices as well as set goals for future discussions.
